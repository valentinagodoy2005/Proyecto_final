Guía para ejecutar el servidor MySQL con Docker

Sigue estos pasos para ejecutar un contenedor Docker que lance un servidor MySQL de prueba:

  1. Paso 1: Asegúrate de tener Docker instalado en tu máquina.
  2. Paso 2: Ejecuta el siguiente comando en tu terminal para lanzar el servidor MySQL: 
    docker run --name mysql-backup-test -e MYSQL_ROOT_PASSWORD=root -e MYSQL_DATABASE=testdb -p 3306:3306 -d mysql:latest
    Esto creará un contenedor con MySQL, con el puerto 3306 abierto en tu máquina local.
  3. Paso 3: Conéctate al servidor MySQL utilizando las siguientes credenciales en tu programa de backups:
    • Host: localhost
    • Usuario: root
    • Contraseña: root
    • Base de datos: testdb
  4. Paso 4: Ahora puedes usar el programa de backups para gestionar esta base de datos de ejemplo.
  5. Paso 5: Cuando hayas terminado, puedes detener y eliminar el contenedor con: 
    docker stop mysql-backup-test && docker rm mysql-backup-test
