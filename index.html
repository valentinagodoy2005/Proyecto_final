Guía para ejecutar el programa con Docker

Si prefieres utilizar Docker para un entorno más controlado y sin problemas de dependencias, sigue los pasos a continuación:

  1. Paso 1: Clona el repositorio de GitHub en tu máquina local: 
    git clone https://github.com/tu_usuario/tu_repositorio.git
  2. Paso 2: Ve al directorio del proyecto: 
    cd tu_repositorio
  3. Paso 3: Construye la imagen Docker: 
    docker build -t mi_proyecto_python .
  4. Paso 4: Ejecuta el contenedor: 
    docker run -it mi_proyecto_python

Con estos simples pasos, tendrás el proyecto ejecutándose dentro de un contenedor Docker, garantizando un entorno limpio y consistente.
